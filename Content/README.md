# Content Structure

These files prepare the site for future admin uploads. For now, the pages are still static HTML, but a future backend/admin dashboard can save content using these same shapes.

- `home.json`: home page hero, mission, and highlight cards
- `sermons.json`: sermon and message cards
- `bible-studies.json`: weekly Bible Study themes, full notes, daily comments, scriptures, and detail page links
- `events.json`: weekly services and church events
- `feasts.json`: appointed feast calendar entries, scripture references, dates, and images
- `ministries.json`: ministry cards and future leader assignments
- `gallery.json`: future media team photo uploads
